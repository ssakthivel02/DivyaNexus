Sources

DivyaNexus aims to use primary texts, public-domain translations where appropriate, and original Tamil or English explanations.

Planned source categories

  • Rig Veda references by mandala, sukta and mantra.
  • Bhagavad Gita references by chapter and verse.
  • Upanishad references by text, section and verse.
  • Deity and temple information from reviewed public references and original summaries.

Editorial policy

Third-party commentary should not be copied directly. App explanations should be rewritten as original educational summaries with clear source references.

Correction requests

If you find a spelling, translation, attribution or source issue, please use the contact page.
